# Documentação do Projeto

Este diretório contém toda a documentação técnica e gerencial do projeto SET (Software Estimation Tool).

## Estrutura da Documentação

### Documentos Principais

- **[Estudo Viabilidade.md](Estudo%20Viabilidade.md)** - Análise de viabilidade técnica, econômica e operacional do projeto, incluindo justificativa para o desenvolvimento da ferramenta e avaliação de alternativas.

- **[Documento Visao.pdf](Documento%20Visao.pdf)** - Documento de visão do produto que descreve o problema a ser resolvido, os stakeholders envolvidos, os objetivos do sistema e os requisitos de alto nível.

- **[Documentacao de Projeto.pdf](Documentacao%20de%20Projeto.pdf)** - Documentação completa do projeto incluindo arquitetura do sistema, decisões de design, especificações técnicas e diagramas detalhados.
